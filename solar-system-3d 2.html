Format

  • Vertical 9:16, 10–15 sec — built for Reels/Shorts/TikTok
  • Fast pacing — visual changes every 1–2s, no dead air

Visual Structure

  • Bold sans-serif text overlays carry the message, not narration
  • Dark, high-contrast background so text pops
  • One idea per screen — quick "reveal" beats, not paragraphs
  • Safe margins so text isn't cropped by platform UI

Pacing Logic

  • Hook in first 1–2s — must stop the scroll
  • 3–5 rapid beats build the idea
  • Payoff/CTA at the end — follow, save, comment

How To Recreate

  • Script first: one line per beat, 5–8 words max
  • Edit vertical-first: CapCut (auto-captions) or Premiere/Resolve, 1080×1920
  • Text treatment: heavy sans font, high contrast, 0.2–0.3s pop-in
  • Cut aggressively: zero dead time between beats
  • Export: 1080×1920 · 30fps · H.264 · under 60 sec
