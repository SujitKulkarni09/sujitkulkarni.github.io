0 · What's a Claude Skill, and Why Do "Crews" Exist?

A flowchart tells you the shape. It doesn't tell you why "skills" became the word everyone in the Claude ecosystem uses in 2026, or why bundlers now package them like a company org chart. A few minutes of context first.

The problem skills solve. A base LLM answers whatever you type, one turn at a time, with no memory of your team's actual playbook. Ask it to "review this NDA" ten times and you'll get ten subtly different reviews — sometimes it checks liability caps, sometimes it forgets to. Before Skills existed, the fix was pasting the same long instructions into every conversation. Skills replaced that with a file Claude reads automatically, once, exactly when the task matches.

How it actually works, mechanically. Anthropic officially shipped Skills as a supported feature in October 2025. Each skill is a folder with a SKILL.md file: a short YAML header (name + description) plus markdown instructions. At startup, Claude scans every installed skill but reads only the header — roughly 100 tokens each — so having 6, 42, or 166 skills installed costs almost nothing until one is actually needed. This is called progressive disclosure, and it's the single design decision that makes large skill libraries practical instead of a context-window disaster.

Where "crews" and "departments" come from. Skills are inherently composable — nothing stops you from installing one for legal work and one for finance side by side. Once the community started publishing dozens of them, organizing by corporate function (the way a real company has departments) became the obvious packaging metaphor. That's what the carousel you're looking at is: a themed re-packaging of that same open SKILL.md format, grouped as if Claude Code were staffing an office.

Where the specific skills in this guide come from. Most of what follows traces back to three real, publicly inspectable sources: Anthropic's own official skills repo, Anthropic's Financial Services and Knowledge Work plugin collections, and a widely-forked community curation, w95/awesome-claude-corporate-skills (166 skills, 14 categories, MIT licensed), which happens to contain almost every skill name shown in your carousel, verbatim.
